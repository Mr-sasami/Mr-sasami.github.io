yilia在火狐时点击触发弹出空页面

原因

浏览器默认处理事件的顺序有差异。
Chrome顺序：onclick -> href -> target
IE和Firefox顺序：onclick -> target -> href

解决方法

添加onclick事件，直接return false阻止之后浏览器默认事件的执行。
